# vanilla-js-todo

JavaScript の基礎学習として作成した、シンプルでミニマルな TODO アプリです。

## 🚀 ライブデモ
[こちらからアプリを試せます](https://mo2lab.github.io/vanilla-js-todo/)

## ✨ 主な機能
- **タスク追加**: 入力フォームから新しいTODOを作成。
- **ステータス管理**: 「未完了」「完了」の切り替えが可能。
- **タスク削除**: 不要になったタスクを一覧から削除。

## 🛠 使用技術
- **言語**: HTML5 / CSS3 / JavaScript (ES6+)
- **構成**: ES Modules (import / export) によるコード分割
- **ツール**: [Vite](https://vitejs.dev/) / StackBlitz

## 📝 こだわり・学習ポイント
- フレームワーク（React等）を使わず、DOM操作をネイティブな JavaScript で実装しました。
- メンテナンス性を考慮し、機能ごとにファイルを分割するモジュール化を実践しました。
- GitHub Pages での公開にあたり、静的ファイルのパス解決やビルドの仕組みを学びました。
