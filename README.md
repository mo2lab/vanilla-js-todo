# Vanilla JS Todo App

JavaScriptの基礎学習として作成した、シンプルでミニマルなTODOアプリです。
フレームワークを使わず、ブラウザ標準の機能（DOM操作）のみで実装しています。

## ライブデモ
[こちらからアプリを試せます](https://mo2lab.github.io/vanilla-js-todo/)

## 学習の背景
このプロジェクトは、JavaScriptのコアな概念を理解するために作成しました。
現在は、このアプリをベースにReactへ移行したプロジェクト「react-todo」も公開しており、ライブラリによる開発効率の違いを比較学習しています。

## 主な機能
- タスク追加: 入力フォームから新しいTODOを作成。
- ステータス管理: 「未完了」「完了」の切り替えが可能。
- タスク削除: 不要になったタスクを一覧から削除。

## 使用技術
- 言語: HTML5 / CSS3 / JavaScript (ES6+)
- 構成: DOM操作による動的なUI更新
- ツール: Vite / StackBlitz

## こだわり・学習ポイント
- フレームワークを使わず、命令的なDOM操作（createElementやappendChild等）を自力で実装しました。
- 機能ごとにファイルを分割するモジュール化を実践し、コードのメンテナンス性を考慮しました。
- GitHub Pagesでの公開を通じ、ビルドプロセスやパス解決の仕組みを習得しました。
